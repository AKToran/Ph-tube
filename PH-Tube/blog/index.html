We can use 'var', 'let' and 'const' to declare variables. The scope of 'var' is 'functional'. A variable declared using var in a block inside a function is accessible in the whole funtion. It leads to problem like having multiple variables with the same name. So we can use 'let' to declare variables which has a 'block' scope. It throws error if we declare a variable with an existig variable name. And a variable declared using 'let' can only be accessed inside the block it is declared. We can reassign its value. So when we need to declare a variable which has a fixed value we use 'const'.

'null': If an initialized variable is not assigned to a meaningful object it throws 'null'. Also if a function does not return any value it throws 'undefined'. 'undefined': If a variable declared but not yet initialized it is 'undefined'.
